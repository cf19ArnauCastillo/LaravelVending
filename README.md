## Software Requerido
   - Npm 
   - Laravel 9.0
   - MariaDB o Mysql
   - Editor, preferiblemente Visual Studio Code

## Como lanzar el proyecto    
    - git clone URL_REPOSITORIO
    - composer install
    - copy .env.example .env
    - DB_HOST=localhost
    - DB_DATABASE=vending
    - DB_USERNAME=arnau
    - DB_PASSWORD=arnau
    - php artisan key:generate
    - php artisan migrate
    - sudo npm install
    - php artisan serve
